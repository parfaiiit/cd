  + 如果您不是从官方得到的本程序,很可能获得的不是最新版本并且无法使用或有很多BUG.

  + 官方推荐服务器配置：php5.2+mysql+zend3，并且支持<?=?>短标记格式

  + 请确认以下目录或文件为可写，linux主机下属性为0777可写：

    /attachment
    /data
    /data/sessions
    /data/caches
    /data/config.db.php
    /data/cron.cache.php
    /backup
	/include/qqlogin/config.inc.php

简单的安装方法: (以下每一步都非常关键,请仔细阅读)

  + 上传目录下所有程序文件到服务器

  + 运行 http://你的域名/install 根据提示安装程序,设置管理员账号
  
  + 系统安装完毕后，请删除install目录，然后进入后台管理进行系统设置，
  然后进行行业类别设置，地区设置，然后再进行其它操作。

  + 系统后台管理路径为：http://你的域名/admin/index.php
